
Desafios 

    1. Adicione os números 1,2,3 em uma variável chamada minhaLista e armazene os números 4, 5 e 6 separadamente em outra variável. Use o método 
       concat() para combinar as sequências de valores em uma nova lista chamada novaLista. Imprima novaLista no console.

    2. Remova o último elemento de novaLista. Imprima novaLista após a remoção.

    3. Utilize o algoritmo de Fisher-Yates (também conhecido como Knuth Shuffle) para embaralhar os elementos em novaLista. Pesquise e adapte o 
       código para realizar o embaralhamento.

    4. Crie uma função chamada removerDuplicatas que aceita um array como parâmetro e retorna um novo array sem elementos duplicados. Teste a função 
       com novaLista e imprima o array resultante.
